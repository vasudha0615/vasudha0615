## 🚀 Featured Project — Internship

<div align="center">

### 🧠 AI-Based Knowledge Graph Builder for Enterprise Intelligence

> *End-to-end AI platform that extracts entities from 8,469 enterprise tickets, builds a dynamic knowledge graph, and answers natural language questions using a RAG pipeline*

[![KGB](https://img.shields.io/badge/View%20Project-Knowledge%20Graph%20Builder-06b6d4?style=for-the-badge&logo=graphql&logoColor=white)](https://github.com/vasudha0615/AI-Based-Knowledge-Graph-Builder-For-Enterprise-Intelligence)
[![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)]()
[![Mistral](https://img.shields.io/badge/Mistral%20LLM-7C3AED?style=flat-square)]()
[![FAISS](https://img.shields.io/badge/FAISS-00A3E0?style=flat-square)]()
[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=flat-square&logo=streamlit&logoColor=white)]()

</div>

```
📦 8,469 Tickets  →  🤖 Mistral LLM  →  🕸️ Knowledge Graph  →  🔍 RAG Q&A  →  📊 Dashboard
      Ingested          Entity Extract      NetworkX + Neo4j      FAISS Search      Streamlit
```

**What I Built:**

| Milestone | What | Tech |
|---|---|---|
| ✅ M1 Data Ingestion | Cleaned & enriched 8,469 tickets (17→21 cols) | Pandas, NumPy |
| ✅ M2 Entity Extraction | LLM-based NER → Knowledge Graph | Mistral, Ollama, NetworkX |
| ✅ M3 Semantic RAG | Vector search + AI Q&A pipeline | FAISS, Sentence Transformers |
| ✅ M4 Dashboard | Glassmorphic enterprise dashboard | Streamlit, Plotly, Flask |
